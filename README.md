secp256k1 C library wrapped with CFFI to use with Python2 and PyPy2.

```
git clone --recursive https://github.com/czepluch/pysecp256k1.git
```

## Development
To build package
```
python setup.py build
```

To install
```
python setup.py develop
```


## Install

To build package
```
python setup.py build
```

To install package
```
python setup.py install
```
